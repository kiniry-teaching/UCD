scenario_chart GAME
indexing 
created "2008-03-10 nor sj cm";

explanation "Each scenario in the game and what happens when they occur."

scenario load screen appears and provides the user with options on how to proceed i.e. new game, saved game, options.
scenario new game starts the game at level one.
scenario saved game starts a previous game where the user last left off.
scenario options gives the user options for example music and sound effects on/off.
scenario ship is destroyed when the ship crashes or is shot down restart the level LIVES decreases by one.If no LIVES remain user is brought back to load screen.
scenario pause game pauses the game to be resumed at a later stage.
scenario ship runs out of fuel the ship can no longer fly and will crash ending the LEVEL and decreasing LIVES by one.
scenario on complete LEVEL the user will proceed to the next LEVEL, LEVEL increases by one.
scenario factory is destroyed triggers a time limit of ten seconds to complete the LEVEL.
scenario enemy is destroyed clears a safe path for the SHIP.
scenario end game can be brought about by the ship running out of lives or by the user completing the last level.
scenario quit the user decides to terminate the game.
end
